## Requirements

- Node.js, version 20.17.0 or above.

## How to run

Make sure your terminal is pointing to:

```bash
../LoanMe/Frontend
```

Ensure dependencies are installed by running:

```bash
npm install
```

Run the local server with:

```bash
npm run dev
```

This application will be pointing to port 3001:

```bash
http://localhost:3001
```
